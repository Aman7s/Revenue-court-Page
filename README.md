# Revenue Courts Landing Page

A modern landing page project built as part of The Odin Project curriculum to practice core frontend development concepts such as Flexbox, layout structuring, image handling, UI styling, and project deployment.

This project is centered around the theme of Revenue Courts and Judicial Hierarchy, presenting information through a clean multi-section landing page design. The goal of the project was not only to build a visually structured webpage, but also to strengthen practical CSS layout skills and understand how real frontend projects are developed and deployed.

## Project Overview

The landing page includes:

Responsive-style navigation header
Hero section with introductory content
Informational court-stage cards
Inspirational quote section
Call-to-action section
Footer

The design follows a clean section-based structure commonly used in modern landing pages and focuses on clarity, spacing, and visual hierarchy.

## Image Handling and Layout Control

The most challenging part of the project was managing images inside flexible layouts.

Large images frequently caused:

overflow issues
layout distortion
inconsistent spacing

## To solve these problems, I learned:

how to resize and optimize images properly
how images behave as flex items
how to fit large images inside smaller containers using CSS
the importance of properties like object-fit, width, height, and container control

This project gave me practical experience debugging real layout issues instead of only learning theory.
